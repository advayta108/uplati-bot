## Изменения в конфигурации TypeScript

### tsconfig.json (основной проект)

**Обновлено:**
- `target`: `es2016` → `ES2022` (современные возможности JavaScript)
- `lib`: добавлен `["ES2022"]` (поддержка новых API)
- Добавлены опции:
  - `allowSyntheticDefaultImports`: true
  - `resolveJsonModule`: true
  - `moduleResolution`: "node"
  - `declaration`: true
  - `declarationMap`: true
  - `sourceMap`: true

### lib/uplati-sdk/tsconfig.json

Аналогичные обновления для библиотеки.
